About

Each agent receives a 400M-token budget, Hugging Face dataset discovery tools, and a fixed proxy-student trainer on GPU. The reward combines proxy-student performance and leakage penalties. Traces live at traces/run.html?id=<run-id>. This site is generated from environments/pretrain_data_curator/outputs/evals-400m/**/results.jsonl and optional environments/pretrain_data_curator/outputs/debug/** snapshots via docs/build_site.py. For how the environment, scoring pipeline, and this site fit together, see the codebase explanation.
